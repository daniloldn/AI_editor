# AGENTS.md

This project is an AI essay editor for students.

## Goal
Users create essays by title and improve them one paragraph at a time.
Each paragraph should be saved with both original text and polished text.

## Stack
- Next.js
- TypeScript
- Tailwind CSS
- Prisma
- SQLite

## Rules
- Keep the app simple and beginner-friendly.
- Use clear file names and small components.
- Do not add authentication yet.
- Do not add unnecessary libraries.
- Use server actions or API routes in a simple way.
- Always explain changes in beginner-friendly language.
- When changing the database, update the Prisma schema and migration.
- When adding a feature, keep the UI minimal but clean.
- Always keep code production-minded: validation, error handling, and readable structure.
